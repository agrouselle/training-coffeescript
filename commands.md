# Coffee commands


```
coffee -c test.coffee   # Compile test.coffee and create test.js
```

```
coffee -wc test.coffee  # Watch and compile test.coffee every time it is updated
```

```
coffee -c src -o js     # Compile all .coffee files in src directory into js directory
```

```
coffee -wc src -o js    # Watch and compile all .coffee files in src into js directory
```
